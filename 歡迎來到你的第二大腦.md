# 歡迎來到你的第二大腦

這個資料夾已經可以直接用 Obsidian 開啟成 vault。

## 如果你在 Obsidian 看不到資料

最常見原因是：Obsidian 開到別的 vault。

請在 Obsidian 選：

`Open folder as vault`

然後選這個資料夾：

```text
/Users/jacob/Projects/2026 codex
```

> [!NOTE]
> 舊 Google Drive 雲端路徑已於 2026-06-12 遷移至本地 /Users/jacob/Projects/... 並停用，防範 AI 走錯路。

## 資料夾結構

| 資料夾 | 用途 |
|---|---|
| `Clippings/` | 原始輸入 |
| `知識庫/` | 整理後的知識 |
| `創作庫/` | 原創輸出 |
| `每日筆記/` | 每日紀錄與週報 |
| `Templates/` | 筆記模板 |
| `docs/` | 專案文件與駕駛艙 |

## 常用說法

| 你說的話 | Codex 會做的事 |
|---|---|
| 「新增到筆記」 | 判斷分類後存入 vault |
| 「消化資料」 | 把 `Clippings/` 整理到 `知識庫/` |
| 「跑一次知識重整」 | 更新索引、操作紀錄與週報 |
| 「開工」 | 讀 AGENTS、駕駛艙、Git/Firebase 狀態 |
| 「收工」 | 更新駕駛艙、提交並推送相關變更 |
